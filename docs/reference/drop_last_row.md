# Remove the octave (last row) from a scale table

Remove the octave (last row) from a scale table

## Usage

``` r
drop_last_row(scale_table)
```

## Arguments

- scale_table:

  A scale table resulting from `ps_scale_table`, `cps_scale_table` or
  `et_scale_table`

## Value

The scale table without the final periodic note
