# Convert a label string to a matrix

Convert a label string to a matrix

## Usage

``` r
label2matrix(label, separator)
```

## Arguments

- label:

  A character vector of labels (e.g., "1x3x5")

- separator:

  Character used to separate harmonics (e.g., "x")

## Value

A matrix where each row is a set of harmonics derived from the label
