# Extract unique harmonics from a set of labels

Extract unique harmonics from a set of labels

## Usage

``` r
label2harmonics(label, separator)
```

## Arguments

- label:

  A character vector of labels (e.g., "1x3x5")

- separator:

  Character used to separate harmonics (e.g., "x")

## Value

A sorted numeric vector of all unique harmonic factors found in the
labels
